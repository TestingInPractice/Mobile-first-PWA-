# Hermes agent. Как ставить задачи агенту? — тезисы


Практические принципы постановки задач AI-агентам:

- Задача должна быть **конкретной и измеримой** — "напиши тесты" это не задача
- Один запрос = одна задача, не смешивай разное в одном промпте
- Агенту нужен **контекст**, не просто команда: покажи файлы, структуру, ожидаемый результат
- Всегда указывай **критерии готовности** — как понять, что задача выполнена
- Используй **цепочку задач**, а не одну большую — разбивай на шаги
- **Верифицируй результат** — агенты склонны говорить "готово" даже при ошибках

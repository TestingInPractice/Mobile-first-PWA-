# DOTI — ИИ-агенты без хаоса в коде: тезисы


DOTI — структурированный инструмент для работы с AI-агентами, основанный на годовом опыте автора. Превращает хаотичные промпты в повторяемый процесс через четыре этапа: **Context → Plan → Execute → Verify**.

**Ключевые принципы:**
- Контекст должен быть явным, не полагайся что агент "сам догадается"
- План — прежде чем генерировать код, агент должен показать что он собрался делать
- Верификация — агент должен сам проверить свою работу, а не говорить "готово"
- DOTI даёт конкретный шаблон, а не абстрактные советы

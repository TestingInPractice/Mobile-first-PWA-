Три примера AGENTS.md файлов для настройки AI coding agents:

**Boris Cherny (расширенный):**
- **Workflow Orchestration (6 правил):** Plan Node Default, Subagent Strategy, Self-Improvement Loop (вести `tasks/lessons.md`), Verification Before Done, Demand Elegance (balanced), Autonomous Bug Fixing
- **Task Management:** план → `tasks/todo.md`, verify → implement → track → explain → document → capture lessons
- **Core Principles:** Simplicity First, No Laziness, Minimal Impact

**Self-Improvement Loop** — запись уроков после каждой ошибки. Ключевой паттерн для continuous improvement AI-assisted разработки.
